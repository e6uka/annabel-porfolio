# Okoye Annabel Adanna - Nursing Portfolio

This folder contains a static HTML portfolio website designed for hosting on any platform
(GitHub Pages, Netlify, Vercel, etc).

## Contents
- `index.html` — Main portfolio webpage  
- `Okoye_Annabel_Adanna_Resume.pdf` — Matching resume PDF  
- `README.md` — Instructions and hosting guide  

## Hosting Instructions

### ➤ GitHub Pages
1. Create a repo named anything you want.
2. Upload the files in this zip into the root of the repo.
3. Go to **Settings → Pages**.
4. Set the branch to `main` and save.
5. Your site will be live shortly.

### ➤ Netlify
1. Go to netlify.com
2. Drag and drop the folder contents into the deploy box.
3. Done.

### ➤ Vercel
1. Import the folder as a new project.
2. Framework = "Other".
3. Deploy.

## Notes
No build step required. Just upload and host.
